# 1
- X = (x1, x2)
- Y = (y1, y2)

- считаем, что с точностью до перестановки:
- (x1 -> x2) && (y1 -> y2) 

- если (x2 -> y1) || (y2 -> x1), то интервалы непересекаются
- - если x2 -> y1, то X -> Y
- - если y1 -> x2 => y2 -> x1, то Y -> X
- если (y1 -> x2) && (x1 -> y2), то интервалы пересекаются

# 2
Доказать:
- иррефлексивность: forall X !(X -> X)
- антисимметричность: forall X forall Y ((X -> Y) => !(Y -> X))
- тразитивность: forall X forall Y ((X -> Y) && (Y -> Z) => (X -> Z))
Доказательство:
- X = (x1, x2), Y = (y1, y2), x1 -> x2, y1 -> y2

- если X -> X, то по определению `->` над интервалами: x2 -> x1. Противоречие 

- если X -> Y, то x2 -> y1
- тогда раз x2 -> y1, то !(y1 -> x2)
- если y1 -> x2, то y2 -> x1
- следовательно из X -> Y => !(y2 -> x1) => !(Y -> X)

- если X -> Y, то x2 -> y1
- если Y -> Z, то y2 -> z1
- значит x2 -> z1 и значит X -> Z
