# 1
- `X = (x1, x2)`
- `Y = (y1, y2)`
- `x1 -> x2` (если не так, выполним перестановку `x1` и `x2`)
- `y1 -> y2` (если не так, выполним перестановку `y1` и `y2`)

- если `(x2 -> y1) || (y2 -> x1)`, то интервалы непересекаются
  - если `x2 -> y1`, то `X -> Y`
  - если `y1 -> x2`, а значит и `y2 -> x1`, то `Y -> X`
- если `(y1 -> x2) && (x1 -> y2)`, то интервалы пересекаются

# 2
Доказать:
1) иррефлексивность: `forall X !(X -> X)`
2) антисимметричность: `forall X forall Y ((X -> Y) => !(Y -> X))`
3) транзитивность: `forall X forall Y (((X -> Y) && (Y -> Z)) => (X -> Z))`

Доказательство:

`X = (x1, x2)`, `Y = (y1, y2)`, `Z = (z1, z2)`, `x1 -> x2`, `y1 -> y2`, `z1 -> z2`
1) если `X -> X`, то по определению: `x2 -> x1`. Противоречие 
2) `X -> Y => x2 -> y1 => !(y1 -> x2) => !(y2 -> x1) => !(Y -> X)`
3) - `X -> Y => x2 -> y1` (а)
   - `Y -> Z => y2 -> z1` (б)
   - `(а) && (б) => x2 -> z1 => X -> Z`
